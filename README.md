##  Instalar localmente

Clonar el projecto de git

```bash
  git clone https://github.com/Universiad-Fidelitas/mayeBeautySalon.git
```

Vamos al directorio clonado

```bash
  cd mayeBeautySalon
```

Instalamos las dependencias

```bash
  npm run install-dependencies
```

Iniciamos el projecto

```bash
  npm run start-dev
```
